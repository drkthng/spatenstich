State rules:

- Compliance section always shown, always grayed (`opacity-50`), never interactive
- After confirm: navigate to Home, show success InlineBanner with count ("3 Entwürfe gespeichert")

## Interaction Contracts

### Inline Validation (Paste-Fallback)

- Trigger: `onBlur` on TextInput (not `onChange`)
- Invalid JSON syntax → InlineBanner variant="warning": "Ungültiges JSON-Format. Prüfe ob der komplette Block kopiert wurde."
- Valid JSON, schema mismatch → InlineBanner: "`schemaVersion` fehlt oder ungültig. [Schema kopieren]"
- Valid JSON, valid schema → clear banner, enable "Importieren" button

### "Schema kopieren" Action

- Copies the contents of `schemas/spatenstich-import.v1.json` to clipboard
- Uses `expo-clipboard` (already in Expo SDK 55 bundle — no new dependency)
- After copy: InlineBanner action text changes to "Kopiert!" for 2 seconds, then reverts
- No modal, no toast — inline feedback only

### Entity Toggle

- Component: `Switch` primitive (react-native Switch wrapped in NativeWind)
- Toggle state stored in local component state, NOT persisted until "Ausgewählte übernehmen" is tapped
- Red-confidence entity toggle: toggling ON shows inline warning text below card "Niedrige Erkennungssicherheit — bitte manuell prüfen"

### Confirm Action ("Ausgewählte übernehmen")

- Writes selected entities to local SQLite Draft tables first (offline-first, Phase 3 pattern)
- Sync to Supabase via existing Outbox SyncWorker
- Loading state: button shows ActivityIndicator + "Wird gespeichert…" text, disabled
- Success: pop back to Home screen; Home shows `InlineBanner` ("N Entwürfe gespeichert — im Editor verfügbar")
- Error: InlineBanner below button "Speichern fehlgeschlagen. Versuche es erneut." + retry button

### Share-Intent Entry Point

- Handled via Expo config plugin (iOS: UTType `public.json`; Android: `intent-filter` for `application/json`)
- Deep link: `spatenstich://import?payload=<base64-encoded-json>` (URL Scheme)
- On launch via share-intent: skip Entry screen, jump directly to Preview screen with parsed payload
- If payload fails validation on launch: show Import Entry screen with error banner pre-populated
