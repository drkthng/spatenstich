# Plan 01: Monorepo + packages/shared + StorageAdapter

> Legt die pnpm-Monorepo-Struktur an, scaffolded `packages/shared` (Types, Constants, Utils, `de.json`) und implementiert den `StorageAdapter` mit Native- und Web-Implementierung inklusive Schema-Versionierung.

## Context

Phase 1 ist reine Infrastruktur. Dieser Plan liefert das Fundament: Monorepo kompiliert, `packages/shared` ist der zentrale Speicher für Types/Konstanten/Utilities/i18n (D-04), und `StorageAdapter` abstrahiert `expo-sqlite` (native) + IndexedDB (web) hinter einem einzigen Interface (D-08). Kein Feature-Code darf `expo-sqlite` jemals direkt importieren — das ist ab Tag 1 verboten (FOUND-02). Schema-Migrationen folgen dem Versions+Up-Pattern (D-09).

Die Pläne 02 (Supabase-Schema) und 03 (EAS CI + Edge Function) bauen darauf auf und können nach Wave 1 dieses Plans parallel laufen, sobald `packages/shared` grün ist.

## Decisions Implemented

| ID | Decision | Task |
|----|----------|------|
| D-04 | packages/shared = Types + Constants + Utils + i18n | 1-01-02 |
| D-05 | Kein geteilter DB-Client | 1-01-01 (Struktur; Clients in Plan 02/03) |
| D-08 | StorageAdapter CRUD only (get/set/delete/list) | 1-01-04 |
| D-09 | Lokale Schema-Versionierung + up-migration-Pattern | 1-01-04 |
| NFR-06 | de.json in packages/shared | 1-01-02 |

#### Task 1-01-04: StorageAdapter-Implementierungen (Native + Web) + Migrations
- **files_modified:** app/src/storage/StorageAdapter.ts, app/src/storage/SqliteAdapter.ts, app/src/storage/IndexedDbAdapter.ts, app/src/storage/migrations.ts, app/src/storage/index.ts, app/src/storage/__tests__/StorageAdapter.test.ts
- **depends_on:** 1-01-02, 1-01-03
- **type:** implementation
- **autonomous:** true
- **requirement:** FOUND-02
- **tdd:** true

**Behavior (tests authored first):**
- `get('missing')` → `null`
- `set('k','v')` then `get('k')` → `'v'`
- `delete('k')`, then `get('k')` → `null`
- `list()` → alle Keys; `list('user:')` → nur Keys mit Prefix
- `getSchemaVersion()` vor Migrationen → 0
- `setSchemaVersion(5)` dann `getSchemaVersion()` → 5
- `runMigrations()` mit einer Migration v1 erhöht Schema-Version auf 1 und ist idempotent bei erneutem Aufruf

**What to do:**

1. **Re-Export des Interface** `app/src/storage/StorageAdapter.ts`:

   ```ts
   import * as SQLite from 'expo-sqlite';
   import type { StorageAdapter } from '@spatenstich/shared';

   const SCHEMA_VERSION_KEY = '__schema_version__';

   export class SqliteAdapter implements StorageAdapter {
     private dbPromise: Promise<SQLite.SQLiteDatabase>;

     constructor(dbName: string) {
       this.dbPromise = (async () => {
         const db = await SQLite.openDatabaseAsync(dbName);
         await db.execAsync(
           `CREATE TABLE IF NOT EXISTS kv (key TEXT PRIMARY KEY NOT NULL, value TEXT NOT NULL);`
         );
         return db;
       })();
     }

     async get(key: string): Promise<string | null> {
       const db = await this.dbPromise;
       const row = await db.getFirstAsync<{ value: string }>(
         'SELECT value FROM kv WHERE key = ?', key,
       );
       return row?.value ?? null;
     }

     async set(key: string, value: string): Promise<void> {
       const db = await this.dbPromise;
       await db.runAsync(
         'INSERT INTO kv(key,value) VALUES(?,?) ON CONFLICT(key) DO UPDATE SET value=excluded.value',
         key, value,
       );
     }

     async delete(key: string): Promise<void> {
       const db = await this.dbPromise;
       await db.runAsync('DELETE FROM kv WHERE key = ?', key);
     }

     async list(prefix?: string): Promise<string[]> {
       const db = await this.dbPromise;
       const rows = prefix
         ? await db.getAllAsync<{ key: string }>('SELECT key FROM kv WHERE key LIKE ?', `${prefix}%`)
         : await db.getAllAsync<{ key: string }>('SELECT key FROM kv');
       return rows.map(r => r.key);
     }

     async getSchemaVersion(): Promise<number> {
       const v = await this.get(SCHEMA_VERSION_KEY);
       return v ? Number(v) : 0;
     }

     async setSchemaVersion(version: number): Promise<void> {
       await this.set(SCHEMA_VERSION_KEY, String(version));
     }
   }
   ```

   ```ts
   export class IndexedDbAdapter implements StorageAdapter {

         upgrade(db) { if (!db.objectStoreNames.contains(STORE)) db.createObjectStore(STORE); },
       });
     }

       const v = await db.get(STORE, key);
       return (typeof v === 'string') ? v : (v ?? null);
     }
     async set(key: string, value: string): Promise<void> {
       const db = await this.dbPromise;
       await db.put(STORE, value, key);

       await db.delete(STORE, key);
     }
     async list(prefix?: string): Promise<string[]> {

       return v ? Number(v) : 0;
     }
     async setSchemaVersion(version: number): Promise<void> {
   ```

   ```ts
   export interface LocalMigration {
     version: number;
     up: (adapter: StorageAdapter) => Promise<void>;
   }

   // Phase 1: initiale Bootstrap-Migration (noch kein lokales Schema nötig,
   // aber der Mechanismus ist aktiv und getestet).
   export const MIGRATIONS: LocalMigration[] = [
     { version: 1, up: async () => { /* Bootstrap — kein Schema außerhalb kv */ } },
   ];

   export async function runMigrations(adapter: StorageAdapter): Promise<void> {
     const current = await adapter.getSchemaVersion();
     const pending = MIGRATIONS.filter(m => m.version > current).sort((a, b) => a.version - b.version);
     for (const m of pending) {
       await m.up(adapter);
       await adapter.setSchemaVersion(m.version);
     }
   }
   ```
