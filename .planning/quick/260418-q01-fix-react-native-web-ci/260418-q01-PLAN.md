# Quick Task 260418-q01: Fix CI — react-native-web missing

## Objective

The CI step `expo export --platform web` fails because `react-native-web` is not installed in the `app/` workspace. Expo requires `react-native-web@^0.20.0` as a peer dependency for web export.

## Tasks

### T-01: Add react-native-web to app/package.json

**Files:** `app/package.json`
**Action:** Add `"react-native-web": "^0.20.0"` to `dependencies` (alongside other `react-native-*` packages)
**Verify:** `react-native-web` appears in `app/package.json` dependencies
**Done:** react-native-web entry present

### T-02: Install dependency

**Files:** `pnpm-lock.yaml`
**Action:** Run `pnpm install` in monorepo root to update lockfile
**Verify:** `pnpm-lock.yaml` references react-native-web
**Done:** lockfile updated, install exits 0
